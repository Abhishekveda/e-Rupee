India's e-Rupee CBDC — already live with 60 lakh users across 17 banks — has the infrastructure to power a cross-border settlement layer that makes international payments instant and near-free. This PoC demonstrates exactly how, and it's built directly on RBI's Payments Vision 2025 framework.

Alignment with RBI policy objectives

Payments Vision 2025

Directly implements RBI's stated goal of using CBDC for cross-border payment processing and settlement, as defined in the June 2022 vision document and Payments Vision 2025.

e-Rupee wholesale pilot

Architecture mirrors the e₹-W pilot structure used for interbank settlements. The bridge relay is architecturally equivalent to the NDS-OM settlement mechanism.

Project Dunbar / mBridge

Implements the same CBDC-to-CBDC bridge concept as BIS Project Dunbar — but as open-source, developer-accessible code that any RBI-authorised bank can clone and evaluate.

FEMA & LRS compliance

Every transfer carries a mandatory FEMA purpose code per RBI Master Direction. The API architecture supports hard enforcement of the $250,000 LRS annual cap per sender wallet.
